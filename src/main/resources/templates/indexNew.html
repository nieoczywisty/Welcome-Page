This project gathers all my knowledge so far. In the back-end I've created spring boot app that handles http requests, calls external Rest API found on https://dane.gov.pl/ and collects data in JSON format. Using unmarshalling I've crated POJO classes corresponding to data recieved from API. Collected data contains information such as: GPS positions, signal quality, number of the vehicle etc. from public transportation company in GdaÅ„sk. Then I wanted to present this data somehow, so I've added Leaflet map with marked positions of all vehicles. You can click on any marker and see what line is it. After pressing the button below a new iFrame is created and loaded with map populated with collected data and new table is created, all using some javascript.
